# avalanche-day3-homework
Proyek ini merupakan aplikasi dApp fullstack yang berjalan di jaringan Avalanche Fuji Testnet. Aplikasi ini memungkinkan pengguna menghubungkan dompet melalui Reown/AppKit, menampilkan data yang diambil langsung dari smart contract, serta melakukan pembaruan data dengan mengirim transaksi ke blockchain.
